Void elements ( self-closing or empty elements)

Void Tag	Purpose
<area>	Defines a clickable area inside an image map. (<area> elements are always nested inside a <map> tag.)
<base>	Specifies the base URL for relative links.
<br>	Inserts a line break.
<col>	Defines properties for columns in a table.
<embed>	Embeds external media like videos and PDFs.
<hr>	Creates a horizontal line to separate content.
<img>	Displays an image.
<input>	Creates an input field in a form.
<link>	Links external resources (like CSS files).
<meta>	Defines metadata for the document.
<param>	Specifies parameters for an <object>.
<source>	Provides multiple media sources for <audio> and <video>.
<track>	Adds subtitles or captions to a video.
<wbr>	Suggests optional word breaks in long words.


1. <area>
The <area> tag defines an area inside an image map (an image map is an image with clickable areas).
<area> elements are always nested inside a <map> tag.
Image Map (coords) Generated by http://www.image-map.net/

2. <base>
The <base> tag specifies the base URL and/or target for all relative URLs in a document.
There can only be one single <base> element in a document, and it must be inside the <head> element.
attr = href, target


Boolean attributes - attributes written without values

To use quote marks inside other quote marks of the same type (single quote or double quote), use character references.  ( &quot;)


Common role Values and Their Uses:
role Value	Meaning & Usage
role="main"	Marks the primary content of the page. Used inside <article>.
role="complementary"	Used for sidebars or additional content that supports the main content. Typically inside <aside>.
role="navigation"	Defines a section for navigation menus (e.g., inside <nav>).
role="banner"	Represents a site-wide header (usually inside <header>).
role="contentinfo"	Marks the footer area with contact info, copyright, etc.
role="button"	Used to make elements behave like a button (e.g., <div role="button">).
role="alert"	For notifications or alerts that require immediate user attention.
